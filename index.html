How to Type Special Scenarios

Here are some common Singlish to Sinhala Unicode mappings:

  • `aa` for `ආ`
  • `ee` for `ඊ`
  • `oo` for `ඌ`
  • `ae` for `ඇ`
  • `nnd` for `ඬ`
  • `nng` for `ඟ`
  • `sh` for `ශ`
  • `th` for `ත`
  • `dh` for `ධ`
  • `ch` for `ච`
  • `kh` for `ඛ`
  • `gh` for `ඝ`
  • `jh` for `ඣ`
  • `ph` for `ඵ`
  • `bh` for `භ`
  • `x` for `ං` (bindu)
  • `N` for `ණ`
  • `L` for `ළ`
  • `K` for `ඛ`
  • `G` for `ඝ`
  • `C` for `ඡ`
  • `J` for `ඣ`
  • `T` for `ඨ`
  • `D` for `ඪ`
  • `P` for `ඵ`
  • `B` for `භ`

For consonants followed by a vowel, simply type the consonant followed by the vowel (e.g., `ka` for `ක`, `ki` for `කි`).

For consonants without a following vowel, an 'al' sign (්) will be automatically added (e.g., `k` for `ක්`).
